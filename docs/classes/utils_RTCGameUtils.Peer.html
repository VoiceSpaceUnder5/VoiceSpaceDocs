Class Peer

책임 : Peer 간 연결을 수립하여 localStream 전송한다. Peer 간 연결을 통해 remoteStream 이 도착하면 AudioElement 에 업데이트한다. dataChannel 을 통해 playerDto 를 받아서 자신의 상태를 계속하여 업데이트한다. dataChannel 을 통해 연결된 peer 에게 me 의 상태를 전송한다.
